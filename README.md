# GIF-Search

Themed GIF Search using Giphy API

### Features
- Topic buttons shows 10 GIFs
- Add custom buttons
- GIF Rating for each image
- GIF images animate upon clicking

### Project
This is a class assigment focused on AJAX and API usage.
